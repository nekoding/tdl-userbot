### About
Telegram Userbot x ChatGPT API

### Usage

- Create API development Cred on [my.telegram.org/](my.telegram.org/)
- Run this command

```sh
cp .env.example .env
```

- Replace the `.env` value with your own configuration
- Run the script

```sh
npm run start
```
